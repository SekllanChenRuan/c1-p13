CodeWars Problem: 'Even or Odd'

Explaination:

The logic is that if a number is divisible by 2 with a remainder of 0 then the number is "Even". Anything that's not divisble by 2 is an "Odd" number. I used the '%' operator to do the calculations for checking if the remainder is equals to 0 or not.
