Leetcode Problem: '2620. Counter'

Explaination:

The first call returns n and then the value of n is incremented by the return n++;. For the second call it will return the previous incremented value that wasnt returned: n + 1, then it will increment again due to return n++; and the incremented value would be n + 2 now but it will only be returned in the next call. This process continues for every new call.
