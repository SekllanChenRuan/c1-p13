HackerRank Problem: 'Day 6: JavaScript Dates'

My submission:

Explaination:

I created an array named daysOfWeek that held the string names for the days of the week: ["Sunday", "Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday"] .

Then I created a new date object with the dateString parameter to get a string value representation of the date and assigned it to a const variable named currentDate.

From there I decided to use CurrentDate.getDay() to get the value of the day of the week with the values in the range of 0-6.

Using the value I got from the .getDay() method for our CurrentDate object, I used it as an index to match my daysOfWeek[] array.

For Example: if CurrentDate.getDay() gave me the value 0, then daysOfWeek[0] would be the string value "Sunday".

So I can just assign the string value of the array index to the dayName variable that was initialized at the beginning and return dayName and it will have the day of the week like "Sunday" returned.
